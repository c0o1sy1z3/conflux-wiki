# Phantom协议



## 生态介绍

Phantom是一个跨链DeFi协议，用于创建合成资产并挖矿，甚至支持发行NFT。Phantom用户可以使用加密货币、NFT和其他数字资产作为抵押品来创建合成资产。通过这一方法，Phantom打破了数字货币资产和传统交易资产之间的障碍，有效降低了普通人参与多元化投资类别的门槛。
